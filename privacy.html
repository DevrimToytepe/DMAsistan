DMAsistan Gizlilik Politikası

Son Güncelleme: 1 Ocak 2024

1. Toplanan Veriler

DMAsistan (DMAgentpro uygulaması), kullanıcıların sosyal medya mesajlarını (Instagram, Facebook Messenger, WhatsApp Business) yönetmek için Meta Graph API kullanır. Bu hizmet kapsamında adınız, erişim belirteçleriniz (token) ve mesaj içerikleriniz güvenli sunucularımızda saklanır.

2. Veri Kullanımı

Toplanan mesaj verileri yalnızca size "Otomatik AI Yanıtı" sağlamak ve mesajlarınızı panelinizde görüntülemeniz için kullanılır. Verileriniz 3. şahıslara satılmaz veya dış amaçlarla kullanılmaz.

3. Veri Güvenliği

Tüm şifreler, erişim tokenları ve API anahtarları veritabanımızda (Supabase Edge) yüksek şifrelemeli Row Level Security (RLS) ile korunmaktadır.

4. Veri Silme ve İletişim

Kullanıcılarımız, istedikleri zaman sistemimizden "Uygulama Bağlantısını Kes" işlemi ile tüm Meta Graph API yetkilerini iptal edebilir ve hesap bilgilerini veritabanımızdan anında tamamen silebilir (Hard Delete). İletişim için: devrim.toytepe@gmail.com
